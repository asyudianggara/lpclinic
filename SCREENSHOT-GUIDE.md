# Screenshot Guide

Gunakan screenshot berikut untuk presentasi portofolio:

1. Landing page publik
2. Halaman login
3. Dashboard admin
4. Tabel gejala
5. Tabel kerusakan
6. Basis pengetahuan
7. Form diagnosa member
8. Halaman hasil diagnosa
9. Laporan hasil diagnosa

Tips:

- Ambil screenshot dalam resolusi desktop dan mobile
- Pastikan data demo terlihat
- Hindari screenshot yang masih menampilkan warning/error
- Gunakan akun admin dan member untuk menunjukkan alur lengkap
